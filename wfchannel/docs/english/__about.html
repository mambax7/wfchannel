WF-Channel

WF-Channel is an easy to use module for Xoops (ver 2.0.5 and higher) to create information pages.

It is a way of managing those pages of information that other modules just don't seem to fit.
You can keep together those 'About Us', 'Refer a Friend', 'Link to Us', Privacy Statements
or just about any type of information that you believe would benefit your users.
This module is therefore kept simple but still offers a lot of interesting options.

Here is a short selection of features offered:

  • easy to use module.
  • allows you to create pages using Xoops codes/smilies and/or HTML tags.
  • simple builtin editor or use of Spaw editor if installed.*
  • allows to import html documents.
  • allows to add images.
  • option to define a publish and expiry date if needed.
  • ...

* if you want to use Spaw as editor then you have to install the spaw module too. It is not included in this release.
