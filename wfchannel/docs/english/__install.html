These instructions are for a fresh install only.

If you want to update an older WF-Channel please select 'Update' instructions.

Remember: It is always a good idea to make a database backup before installing any modules.

Fresh installation of WF-Channel

  1. Upload the module to your website

    Upload the 'wfchannel' folder to {xoops-rootdirectory}/modules folder

  2. Change and verify folder permissions

    CHMOD the following folders to 777:

    wfchannel/cache
    wfchannel/images
    wfchannel/images/linkimages
    wfchannel/html
    wfchannel/html/images

  3. Install the module

    Login as administrator and enter Xoops Administration page. Select System --> modules and install wf-sections

  4. Set module permissions

    Most important step now will be to setup the group permissions for the module via Xoops: System --> groups
